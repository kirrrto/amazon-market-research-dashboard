# 亚马逊市场调研仪表盘

## 这个工具适合谁

该工具面向：

- 亚马逊产品开发工程师
- 跨境电商产品经理
- 安防监控产品开发人员
- 车载影像产品开发人员
- 采购、供应链和项目管理人员
- 中小型硬件品牌团队

它的目标不是替代 Helium 10、Jungle Scout、Keepa 等专业数据工具，而是把不同来源的数据统一成一个适合产品开发决策的工作流。

## 核心工作流

### 1. 表格导入

当你已经有调研表或第三方工具导出表时，使用这个模式。

支持：

- CSV
- XLSX
- XLS

系统可以：

- 选择 Excel 工作表
- 自动推荐中英文字段映射
- 人工修正字段映射
- 识别重复 ASIN
- 识别缺失和异常数据
- 导出标准化 Excel 工作簿

### 2. 产品页面连接器

当你有供应商官网或品牌官网的公开产品页面时，使用这个模式。

系统可以提取：

- 产品标题
- 品牌和型号
- JSON-LD Product 数据
- HTML 规格表
- 文档链接
- 抓取日志
- 原始规格
- 初步标准化规格

### 3. 模板中心

当你还没有规范调研表时，先从模板中心开始。

可下载：

- 市场调研模板
- 供应商报价模板
- 产品 URL 导入模板
- 安防摄像头规格模板
- 车载影像规格模板

每个模板都包含：

- 主数据 Sheet
- 数据字典
- 示例 Sheet

### 4. 规格矩阵和缺口分析

当系统已经提取并标准化规格后，可以查看：

- 规格矩阵
- 覆盖率汇总
- 缺口分析

这可以帮助你判断哪些产品参数比较完整，哪些参数需要继续向供应商确认。

## 不支持的内容

当前工具不会：

- 把 Amazon 商品页 HTML 抓取作为核心功能
- 绕过登录
- 绕过验证码
- 访问私有页面
- 绕过付费墙或网站访问限制
- 高频抓取网站
- 自动验证供应商宣传是否真实

## 推荐使用方式

1. 打开在线 Demo。
2. 选择“模板中心”。
3. 下载市场调研模板或产品 URL 模板。
4. 填写或收集一小批样本。
5. 上传表格或粘贴产品页面 URL。
6. 查看问题记录和标准化结果。
7. 查看规格矩阵、覆盖率汇总和缺口分析。
8. 导出 Excel 给团队复核。

## 本地运行

创建虚拟环境：

```bash
python -m venv .venv
```

Windows 激活：

```bat
.venv\Scripts\activate
```

安装依赖并启动：

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```
